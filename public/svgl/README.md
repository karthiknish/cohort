| `facebook-icon.svg` | Facebook |
| `instagram-icon.svg` | Instagram |
| `tiktok-light.svg` / `tiktok-dark.svg` | TikTok (theme variants) |
| `twitter.svg` | Twitter |
| `x-light.svg` / `x-dark.svg` | X (theme variants) |
| `youtube.svg` | YouTube |
